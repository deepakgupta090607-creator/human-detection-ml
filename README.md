# 🧍 Human Action Detection System

## 📌 Overview

This project focuses on detecting human actions using computer vision techniques in Python. It processes images or video streams and identifies human presence with bounding boxes.

## 🚀 Features

* Human detection in images/videos
* Real-time processing
* Bounding box visualization
* Lightweight implementation

## 🛠 Tech Stack

* Python
* OpenCV
* NumPy

## ⚙️ How to Run

```bash
python Human_Action_Detection.py
```
